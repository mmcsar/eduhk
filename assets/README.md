# Assets Directory

This directory contains app assets such as images, fonts, and other media files.

## Required Assets

For the app to work properly with Expo, you need to add the following image files:

### 1. **icon.png**
- Size: 1024x1024 pixels
- Format: PNG with transparency
- Use: App icon for both iOS and Android

### 2. **splash.png**
- Size: 1242x2436 pixels (or larger)
- Format: PNG
- Use: Splash screen displayed when app launches

### 3. **adaptive-icon.png** (Android)
- Size: 1024x1024 pixels
- Format: PNG with transparency
- Use: Android adaptive icon (foreground)

### 4. **favicon.png** (Web)
- Size: 48x48 pixels (or larger)
- Format: PNG
- Use: Browser favicon for web version

## How to Add Assets

1. Create or design your images based on the MMC logo
2. Save them in this directory with the exact names mentioned above
3. Restart the development server

## Temporary Placeholders

If you don't have these assets yet, Expo will use default placeholders. However, for production, you should replace these with branded MMC assets.

## Design Guidelines

- Use the MMC brand colors (Blue #1565C0 and white)
- Include the MMC logo prominently
- Keep designs simple and professional
- Ensure high resolution for all platforms
