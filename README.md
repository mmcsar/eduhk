# eduhk
Application éducative complète pour la gestion des écoles en RDC (attendance RFID/NFC, chat parent ↔ école, calendrier, bulletins, dashboard admin). Développée avec React + Node.js/MongoDB scalable.

## MMC Logo demos
- `mmc-logo` : application Expo React Native qui recrée le sceau MMC via `react-native-svg`.  
  ```bash
  cd mmc-logo
  npm install
  npx expo start
  ```

- `mmc-logo-web` : version web React + Vite + Tailwind CSS avec animation du sceau et bouton de relance.  
  ```bash
  cd mmc-logo-web
  npm install
  npm run dev
  ```
